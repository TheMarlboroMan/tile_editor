= HELP =

== EDITOR MODE ==

f1 - help.
f2 - layer settings
s - save current file
lctrl+s - save to new file
l - load new file
+ - zoom in
- - zoom out
pagedown - next layer
pageup - previous layer
space - change layer drawing mode
tab - toggle toolset 
arrows - move / change toolset element (depending on toolset visibility)

??? - new layer (perhaps its own controller)
??? - delete layer (perhaps its own controller)

=== TILE EDITOR MODE ===

left click - add / change tile
right click - select tile type under cursor
del + click - delete tile
lshift + click - between two points, add/change/delete area
